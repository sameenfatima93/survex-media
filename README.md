# Survex Media - Digital Marketing Agency Website

A modern, responsive, and conversion-focused website developed for **Survex Media**, a digital marketing and software development agency.

The website is designed with a clean, premium UI to establish trust, highlight services, and generate quality leads for the business.

## 🌐 Live Demo

https://survexmedia.netlify.app/

---

## 📌 Project Overview

Survex Media offers a complete range of digital solutions for businesses. This website was designed and developed to provide an engaging user experience while showcasing the company's expertise and services.

The primary objective of the project was to create a professional online presence that reflects the brand identity and improves customer conversion.
